# YSFlight Community Edition (YSCE) mod version 1.0

This is a personal modification based on YSCE.

## Updates

- Improved aircraft bouncing behavior on downward-sloped runways
- Improved tailstrike detection on sloped runways
- Enhanced ground braking force
- Fixed an issue where "6" could not be selected in the ILS/VOR selection screen
- Added game speed adjustment feature  
  - `-` key: 0.5x speed  
  - `+` key: 2.0x speed

## Build

A Binary build for Apple Silicon Mac can be downloaded from
https://github.com/sekiuhn-tech/ysflight-mod/releases/tag/v1.0
